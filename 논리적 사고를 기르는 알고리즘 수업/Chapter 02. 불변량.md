# Chapter 02. 불변량

- '불변'은 '변하지 않음'을 의미하는 한자어다. 비슷한 용어로는 '상수'와 '패턴'이 있다.

## 2.1 초콜릿

### 2.1.1 풀이

1. 초콜릿을 한 번 쪼갤 때마다 항상 초콜릿을 쪼갠 총 횟수는 1만큼, 초콜릿 조각 수도 1만큼 증가한다.
2. 여기서 변하지 않는 값은 바로 쪼갠 횟수와 조각의 수의 차이이다.
3. 그러므로 하나의 초콜릿을 작은 정사각형 조각들로 나누기 위해서는, 정사각형 조각의 총 개수보다 하나 적은 횟수만큼 초콜릿을 쪼개야 한다.

### 2.1.2 수학적 풀이

#### 추상화

- 최고의 문제 해결자는 아마도 추상화에 가장 능숙한 사람일 것이다.

#### 대입

- 문제 풀이의 다음 단계는 초콜릿을 쪼개는 과정을 모델링하는 것이다. 이 과정에서 대입문을 사용한다.
- p,c := p = 1, c + 1
- 대입 기호 ':='는 '좌변이 우변이 된다'라고 읽는다.

#### 귀납

- p - c는 불변량이다. 따라서 초콜릿을 얼마나 쪼갰는지에 관계없이 p - c = 1이다.
- 초콜릿을 작은 정사각형 조각들로 완전하게 나누면 p = s다. 여기서 s는 정사각형 조각의 총 개수이다. 즉, 쪼갠 횟수를 c라고 한다면 s - c = 1이 성립한다. 고로, c = s - 1이다.
- 여기에는 수학적 귀납법이라는 중요한 원리가 사용되었다.

## 2.2 빈 상자

> 문제를 풀기 위한 전략
> 1. 최종 상태에 대해 알려진 것과 알려지지 않은 것을 확인한다.
> 2. 임의의 시점에서의 상태를 나타낼 수 있는 변수를 도입한다.
> 3. 상자에 상자를 넣는 과정을 변수에 값을 대입하는 과정으로 모델링한다.
> 4. 이 과정에서 불별량을 찾는다.
> 5. 위 과정을 결합해 최종 상태를 유도한다.
